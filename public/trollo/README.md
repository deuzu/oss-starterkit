Trollo
======


Technos
-------

  - php 5.*
  - Bootstrap
  - apache2
  - MySQL


Homepage
--------

List all cards in a table

Table with the following columns:
  - id
  - title
  - description
  - actions (edit, supprimer)


Card creation Page
------------------

Form with the following fields:

  - title
  - description

A `create` button and a `cancel` button


Card edition Page
------------------

Form with the following fields:

  - title
  - description

A `save`, `cancel` and a `delete` button.
